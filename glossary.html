Technical Glossary

20 technical terms compiled from the conference and applied to FrontEnd Software Development.

Reglas básicas que rigen la mezcla de colores para conseguir efectos estéticos y perceptuales específicos en medios visuales y pantallas digitales.

Representación de una imagen donde cada píxel tiene un valor de luminancia pura sin información de color, usada para medir contrastes estructurales.

Fenómeno donde la percepción de un tono se altera dependiendo directamente de la tonalidad o color del fondo sobre el cual se sitúa.

Práctica de diseño y desarrollo que garantiza que los sitios web sean usables por cualquier persona, incluyendo aquellas con discapacidades visuales o daltonismo.

La propiedad que define el estado puro de un color en el espectro visible, permitiendo diferenciar un color de otro (ej. verde de rojo).

Nivel de pureza o intensidad de un color específico. Un color altamente saturado es vivo, mientras uno de baja saturación se acerca al gris neutro.

Indica qué tan claro u oscuro es un tono basándose en la cantidad de luz que refleja, variando desde el blanco absoluto al negro absoluto.

Parejas de colores que se sitúan en posiciones diametralmente opuestas en el círculo cromático, generando el máximo contraste e impacto visual al juntarse.

Colores que se encuentran continuos o vecinos en el círculo cromático, compartiendo un tono base y generando composiciones armónicas y tranquilas.

Medida matemática de la diferencia de luminancia entre el texto de primer plano y su fondo, regulada bajo las pautas internacionales WCAG.

Conjunto de elementos arquitectónicos, visuales y pantallas con las que un usuario interactúa directamente dentro de una aplicación o página web.

Proceso integral enfocado en optimizar la facilidad de uso, la lógica de navegación y la satisfacción que experimenta un usuario al operar un sistema informático.

Filosofía de desarrollo frontend que busca que las interfaces se redimensionen e interactúen perfectamente en cualquier tamaño de pantalla, móvil o escritorio.

Modelo de producción que prioriza compartir, reutilizar, reparar y reciclar materiales existentes la mayor cantidad de veces posible para reducir el impacto ambiental.

Campo de estudio que analiza las respuestas conductuales, emocionales e inconscientes que evocan los distintos tonos cromáticos en el ser humano.

Biblioteca frontend de código abierto basada en HTML, CSS y JS utilizada para acelerar el desarrollo responsive estructurando las interfaces mediante rejillas y componentes preestablecidos.

Acción realizada mediante JavaScript para alterar estructural o estéticamente los nodos del documento HTML en tiempo de ejecución sin necesidad de refrescar el navegador.

Composición estética ejecutada utilizando un único matiz o color base, alterando únicamente sus valores de saturación y luminosidad para generar sobriedad.

Conjunto de elementos visuales, tipográficos y cromáticos consistentes que una organización utiliza para proyectar su personalidad ante el público.

Evolución tecnológica de las hojas de estilo que introdujo soporte nativo para bordes redondeados, degradados, animaciones avanzadas y filtros directos sobre las interfaces web.
